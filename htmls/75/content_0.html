ООО «РЭНЕРА» - интегратор Госкорпорации «Росатом» по системам накопления энергии (входит в Топливную компанию «ТВЭЛ»)

C 2012 года ООО «РЭНЕРА» производит порошок катодного материала для литий-ионных аккумуляторов — это единственное в России промышленное производство литированного оксида кобальта (LiCoO2)

Литий-ионные аккумуляторы с применением кобальтата лития применяются в изготовлении легких, портативных изделиях, отличающихся высокой удельной емкостью

ЛИТИЙ КОБАЛЬТ ДИОКСИД

Литий кобальт диоксид (литированный оксид кобальта), химическая формула (LiCoO2), представляющий собой порошок черного или темно-синего цвета, предназначенный для использования в промышленном производстве литий-ионных аккумуляторов.

Перспективная
продукция

Литированный оксид никель-кобальт-алюминия LiА[NiYCoZAlX]O2 (типа NCA)

Проводится работа по подтверждению потребности в данном типе катодного материала производителей литий-ионных аккумуляторов в РФ и гарантированной собственной потребности. Прорабатывается возможность разработки технологии.

Литий железофосфат LiFePO4 (типа LFP)

литий-никель-марганец-кобальт-оксид LiА[NiYMnXCoZ]O2 (типа NMC)

Запланировано создание промышленного производства указанных типов порошков катодного материала для обеспечения перспективной потребности собственного производства литий-ионных аккумуляторов, а также потребности других производителей литий-ионных аккумуляторов в РФ.

Порошок электродных материалов составов Li4Ti5O12 (типа LTO)

Изучается потребность в порошках электродных материалов литий-титаната с целью создания промышленного производства.

Порошок электродных материалов для натрий-ионных аккумуляторов типа фторидофосфат ванадия-натрия (типа NVPF)

орто-пирофосфат железа-натрия (типа NFePP)

Изучается потребность в порошках электродных материалов для натрий-ионных аккумуляторов с целью создания промышленного производства прорабатывается возможность разработки технологии промышленного производства.





Совместно с Институтом химии твердых тел и механохимии СО РАН и Институтом неорганической химии СО РАН прорабатывается возможность проведения НИОКР создания технологии производства порошков катодных материалов для литий-ионных аккумулятоторов и электродных материалов для натрий-ионных аккумуляторов
